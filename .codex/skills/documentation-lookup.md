# documentation-lookup

Origin: adapted from Everything Claude Code (ECC)

Use when:

- answering framework or library questions
- relying on up-to-date API behavior
- touching Flask, SQLAlchemy, Postgres, Telegram APIs, Docker, or similar external surfaces

Core rules:

- prefer live official docs over memory when behavior may have changed
- verify version-sensitive claims
- do not invent undocumented API behavior
- cite primary documentation when it materially affects implementation
