# Project-local Codex Skills

This directory contains a curated subset of skills inspired by Everything Claude Code (ECC).

Why curated, not full import:

- the full ECC surface is very large
- much of it is irrelevant noise for `legal_digest`
- project-local relevance matters more than raw volume
- keeping this set lean lowers context drift for both human operators and AI tools

Current imported/adapted skills:

- `python-patterns`
- `postgres-patterns`
- `database-migrations`
- `search-first`
- `documentation-lookup`
- `frontend-design`
- `security-review`
- `tdd-workflow`
- `terminal-ops`

If more ECC skills become useful later, add them here selectively.
